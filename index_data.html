THE DATA

The data was mainly gathered from two different sources.

Character data

Obtained from the Office fandom wiki, Dunderpedia. We downloaded all character pages using the fandom wiki API obtaining a description of each character. We used this data to make the network by connecting characters that had hyperlinks between them.

Details
  • 297 different character pages
  • .txt files

Famous quotes

Obtained from the Office fandom wiki, Dunderpedia. On Dunderpedia there are famous quotes from each episode of each season. We used these quotes to make a wordcloud for each season.

Details
  • 185 quote files from episodes
  • .txt files

Episode transcript

Obtained from a reddit thread. This is a dataset where where each line corresponds to a sentence that a character says. Each line is identified by what character says the sentence, the season number, episode number and even the scene number. This was put together by a user on reddit, where he looked at the transcripts of each episode and put it together into one csv and characterizing it as described above. There are some typo's in the dataset which we fixed and there may be more typos we did not notice.

Cointains
  • 192 observations
  • 7 variables
